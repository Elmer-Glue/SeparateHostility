## Separate Hostility

Originally made as an update to [Please Just Fight](https://github.com/shibdib/SPT-PleaseJustFight), however it faces a core limitation in 0.16: same-faction PMC groups won't fight each other, only engaging opposing factions. Everyone lives in a big happy family.

To fix this, the mod was fully rewritten with patches that:

Split PMC groups into individual units (per bot and spawn group).

Remove faction-based hostility restrictions.
Result: PMCs now engage any hostile target, including former allies, enabling true free-for-all combat, while keeping their spawn group settings.
